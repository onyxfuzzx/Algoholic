## Installation
To set up the project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/onyxfuzzx/Algoholic.git
   cd Algoholic
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the project**
   ```bash
   npm start
   ```
4. Open your browser and navigate to `http://localhost:3000`
